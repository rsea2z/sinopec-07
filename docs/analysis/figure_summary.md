# 图表与结果总结

本轮已经围绕全部实验结果生成了更完整的论文图表资产，图表输出位于 `results/figures/`，同时生成了论文汇总表 `results/metrics/thesis_summary_table.csv`。所有图表统一采用中文宋体族字体，适合直接插入毕业论文模板。

## 图 1：全部任务最终策略热力图

`figure1_strategy_heatmap` 展示了每个品种在 `T+5 / T+10 / T+20` 三个预测期上的最终入选策略。图中最重要的信息不是“哪种复杂模型最多”，而是“朴素持久性仍然覆盖了大多数任务”。这张图适合支撑全文的核心论点：不存在一个统一强模型可以稳定覆盖全部能源化工期货任务。

## 图 2：具有实质意义的增益任务

`figure2_meaningful_gains` 仅保留相对 `naive persistence` 具有明显 RMSE 改善的任务。当前真正值得保留的任务主要只有五个：

- `PR @ T+5`
- `PR @ T+10`
- `PX @ T+5`
- `PX @ T+10`
- `PX @ T+20`

这张图说明，本文最可靠的结论不是“复杂模型整体优于简单模型”，而是“少数品种在特定预测期上存在可被利用的结构性信号”。

## 图 3：PX 方法对比

`figure3_px_method_comparison` 比较了 PX 上多类专项方法，包括专项搜索、元集成、注意力融合和竞赛式集成。图中能够直观看到，PX 是当前最稳定、最值得重点分析的正例，尤其在 `T+10` 和 `T+20` 上表现突出。

## 图 4：PR 方法对比

`figure4_pr_method_comparison` 展示了 PR 上不同路线的相对表现。与 PX 相比，PR 的可预测性范围更窄，但在 `T+5` 和 `T+10` 上仍形成了稳定的非基线增益，适合作为论文中的第二个专项建模案例。

## 图 5：赛道与预测期增益分布

`figure5_track_horizon_gain` 从赛道层面对结果进行再聚合，展示三个赛道在不同预测期上的平均增益。结果表明，赛道二即聚酯产业链，是当前最有建模价值的赛道，说明产业链内部的中游化工品更可能携带可学习的结构信息。

## 图 6：数据覆盖区间

`figure6_data_coverage` 展示了三个赛道主要品种的样本覆盖时间范围。该图能够直接解释为什么统一复杂模型难以稳定生效，因为不同品种上市时间与可训练样本长度差异较大，特别是 PX 和 PR 的可用历史显著更短。

## 图 7：预测案例图

`figure7_prediction_cases` 给出了 PX 与 PR 的代表性预测案例。其价值在于把“指标提升”转化为更容易理解的时序对比，说明专项模型在某些阶段能够更好跟踪趋势或减少偏差，而不是只停留在表格指标层面。

## 图 8：最终策略分布

`figure8_strategy_distribution` 统计了最终策略板中各类方法的占比。该图进一步强调，最终系统并不是由复杂模型主导，而是由“朴素基线为主、少量专项模型增强”的保守策略构成。

## 图 9：研究流程图

`figure9_workflow_diagram` 将本文的研究流程整理为一张完整流程图，覆盖数据审计、特征工程、模型实验、专项优化、结果汇总和论文编撰。该图适合作为方法章节的总览图，用于提升论文结构表达的完整性。

## 图 10：全部任务效果排名图

`figure10_task_ranking` 是一张最直接、最容易理解的结果图。它把 27 个任务按照相对 `Naive` 的 RMSE 改进值从好到坏排开，绿色表示优于基线，橙色表示不如基线。该图非常适合放在实验结果章节前半部分，用一句话就能支撑核心结论：只有极少数任务真正超过了朴素基线。

## 图 11：Naive 与最终策略逐任务对比

`figure11_final_vs_naive` 将每个任务上的 `Naive` RMSE 与最终策略 RMSE 连接起来。黑点是基线，彩色点是最终策略，连线方向可以直观看到误差是下降还是上升。这张图比单纯表格更能展示“有些任务提升很多，有些任务反而退化”的事实，适合作为补充分析图或答辩展示页。

## 图 12：各赛道胜负分布

`figure12_win_loss_summary` 用堆叠柱状图汇总三个赛道中“优于 Naive”和“不如 Naive”的任务数量。它的优点是非常简洁，适合在摘要式分析里快速说明：赛道二明显更有建模价值，而赛道一和赛道三大多数任务仍应保守回退到基线。

## 图 13：五个专项任务卡片图

`figure13_best_task_cards` 将当前最值得保留的五个专项任务做成卡片式展示，包括品种、预测期、增益和推荐策略。这张图特别适合本科论文和答辩场景，因为它把最终结论压缩成一页就能说清的决策结果。

## 可直接写入论文的总体结论

结合当前图表与结果汇总，可以形成一条较为稳健的论文主线：

第一，能源化工期货价格预测存在极强的价格持久性，`naive persistence` 是必须严肃对待的强基线。第二，统一复杂模型无法在全部品种和全部预测期上稳定超过该基线。第三，真正有效的改进来自少数任务上的专项建模，其中 `PX` 是最强正例，`PR` 是第二正例。第四，最终最优解不是单一模型，而是一张按品种、按预测期分配的策略板。第五，对于论文主文，最推荐优先使用的是图 1、图 2、图 6、图 10 和图 13，这几张图最简洁也最容易被读者快速理解。
