# 图表与结果总结

本轮已基于当前全部实验结果生成论文可复用图表，图表文件输出在 `results/figures/`，同时生成了可直接引用的汇总表 `results/metrics/thesis_summary_table.csv`。所有图表统一使用中文宋体，便于后续直接嵌入论文模板。

## 图 1：全任务策略热力图

`figure1_strategy_heatmap` 展示了每个品种在 `T+5 / T+10 / T+20` 上相对于 `naive persistence` 的最优 RMSE 改进值。颜色越偏红，说明专项策略越有效；越偏蓝，则说明复杂模型反而劣于基线。

这张图最重要的结论是：大多数任务仍然围绕零附近波动，说明不存在覆盖全市场的统一强模型。真正显著的正收益主要集中在聚酯产业链中的 `PX` 和 `PR`，而原油链、塑料链大多应保守回退到 `naive persistence`。

## 图 2：具有实际意义提升的任务

`figure2_meaningful_gains` 只保留改进值大于 `1.0` 的任务，从而把“统计上略优”与“实际可用”区分开。按当前结果，真正值得保留的专项任务只有五个：

- `PR @ T+5`
- `PR @ T+10`
- `PX @ T+5`
- `PX @ T+10`
- `PX @ T+20`

这说明本课题最合理的结论不是“复杂模型整体优于简单模型”，而是“少数品种在特定 horizon 上存在可利用的结构性信号”。

## 图 3：PX 建模路线比较

`figure3_px_method_comparison` 对比了 `PX` 上四类代表性路线：

- 专项集成
- 元集成
- 注意力融合
- 竞赛式集成

从图中可以直接看到，`PX` 是目前最稳定、最值得深入研究的品种。尤其在 `T+20` 上，`px_meta_ensemble` 获得了最大幅度的改进；在 `T+10` 上，竞赛式集成依然最优；在 `T+5` 上，则是 `meta ensemble` 明显强于旧版专项模型。

这组结果很适合在论文中支撑一个重要论点：对同一品种，不同预测期需要不同的建模策略，不能假设存在单一最优模型。

## 图 4：PR 建模路线比较

`figure4_pr_method_comparison` 展示了 `PR` 上不同实验路线的表现。与 `PX` 相比，`PR` 的可预测性范围更窄，主要集中在短中期。

当前最可保留的结果是：

- `PR @ T+5`：`pr_specialist`
- `PR @ T+10`：`residual_search`
- `PR @ T+20`：仍应回退到 `naive persistence`

这说明 `PR` 虽然不如 `PX` 稳定，但已经足以支持论文中“专项建模优于全局统一建模”的第二个案例。

## 可直接写入论文的总体结论

从当前全部图表和汇总表可以形成一条比较完整且稳健的论文主线：

第一，能源化工期货价格预测存在极强的持久性特征，`naive persistence` 是必须严肃对待的强基线。第二，统一复杂模型无法在全部品种和全部 horizon 上稳定超过该基线。第三，真正有效的改进来自产业链内部的局部结构信号，需要按品种、按 horizon 做专项建模。第四，在本项目数据上，`PX` 和 `PR` 是最有代表性的两个正例，其中 `PX` 的中长期信号最强，`PR` 的短中期信号更明显。
